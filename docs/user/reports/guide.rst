.. _reports:

Reporting & SQL Queries
=======================

QATrack+ currently has two different methods of generating reports.  There is a
set of predefined reports in the :ref:`Reports App <reports-reports>` and also a
method of writing and saving SQL queries, generating charts & pivot tables, and
exporting CSV data using the :ref:`Query Tool <reports-query_tool>`.
